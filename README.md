## Установка

<h3 id="install"></h3>
1. Склонировать проект на свой компьютер

```bash
git clone https://github.com/ShibashovNS/react-stellar-burger.git
```

2. Перейти в папку с проектом и установить зависимости в проекте

```bash
cd stellar-burger
npm install
```

3. Запустить проект

```bash
npm start
```
